# Bioluminescence Explorer

Simple static GitHub Pages site about bioluminescence.

## Files

- `index.html` contains the page structure
- `styles.css` contains the design and animation
- `script.js` contains the interactive animal selection and reaction pulse

## Publish on GitHub Pages

1. Push these files to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select your main branch and the `/ (root)` folder.
6. Save, then wait for GitHub Pages to publish the site.

After that, GitHub will give you a public URL for the project page.
